# BabyCam Releases

This repo holds only BabyCam's release assets (`app-release.apk` + `version.json`) for the app's
in-app update feature — no source code lives here. The app itself is at a private repo.

Each release has two fixed-name assets:
- `app-release.apk`
- `version.json` — `{ "versionCode": N, "versionName": "...", "mandatory": bool, "notes": "..." }`
